# Usage #

Interpreter:

```
% horns
```

Script mode:

```
% horns script.lsp
```

# Example #

```
% horns
Horns v0.0.4 (c) 2010 Andrew Pennebaker www.yellosoft.us
Press Control-C to exit.
> (print "Hello World")
Hello World
> (+ 1 1)
=> 2
> (** 2 5)
=> 32
```
